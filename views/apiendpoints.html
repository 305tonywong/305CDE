Member Registration

URI       https://127.0.0.1:3000/registration
Method	  POST
Data	  {
              "username": "YourUserName",
              "email": "your@email.address",
              "password": "YourPassword"
          }
Result	  {
              "Registration": "OK"
          }
Error	  {
              "Registration": "fail"
          }
